Welcome Webmaster!

Welcome to your new ImpressCMS powered website. If you haven't already, please delete the install folder from the server and ensure that mainfile.php is not writeable (chmod 444).

To begin administering your new ImpressCMS powered website you can click the Administration Menu link located on the left of this page.

You may want to begin by editing your website Preferences: In the admin panel, hover over the System dropdown and select Preferences.

Afterwards you can begin adding Modules and Themes.

Many of the available modules and themes for ImpressCMS, are available at the Addons section of the projects website.

You will also need to begin using Blocks. You can begin by removing this block. You can do this by navigating to System Admin > Blocks, and the selecting "Webmasters" in the Groups select box. You will then be able to see the blocks available for the Webmasters group, which this block is!

For more information about working with ImpressCMS, please use the links below.

  • Using ImpressCMS
  • Customizing ImpressCMS
  • Developing for ImpressCMS

We warmly invite you to join The ImpressCMS Community - Where you can make contributions, get help, help others, etc...
